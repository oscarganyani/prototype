To Run this program please install the following packages

Mac Command 
    - pip3 install flask flask_sqlalchemy flask_login flask_bcrypt flask_wtf wtforms email_validator

Windows Command 
    - pip install flask flask_sqlalchemy flask_login flask_bcrypt flask_wtf wtforms email_validator

linux command 
    - pip install -U Flask-SQLAlchemy Flask-Login Flask_Bcrypt Flask_wtf wtforms Email_Validator

 NB**Note the authentication part of the code is not ours. We followed a youtube tutorial thus we DO NOT take any credit
 for the code.
